Learn Flask: A Comprehensive Roadmap

1. Setting Up the Flask Environment

Install Flask and set up a virtual environment:

pip install Flask
python -m venv venv
source venv/bin/activate  # On Windows use venv\Scripts\activate

Create a simple Flask app:

from flask import Flask

app = Flask(__name__)

@app.route('/')
def hello_world():
    return 'Hello, World!'

if __name__ == '__main__':
    app.run(debug=True)

Run the app:

python app.py

2. Flask Routing and URL Building

Add basic routes:

@app.route('/')
def home():
    return 'This is the homepage!'

@app.route('/about')
def about():
    return 'This is the about page.'

@app.route('/user/<username>')
def show_user_profile(username):
    return f'Hello, {username}!'

4. Handling Forms and User Input

Create a form in templates/form.html:

<!DOCTYPE html>
<html>
<head>
    <title>Form Example</title>
</head>
<body>
    <h1>Submit Your Name</h1>
    <form method="POST">
        <label for="name">Name:</label>
        <input type="text" id="name" name="name">
        <input type="submit" value="Submit">
    </form>
</body>
</html>

Handle the form submission:

@app.route('/submit', methods=['GET', 'POST'])
def submit():
    if request.method == 'POST':
        name = request.form['name']
        return f'Thank you, {name}!'
    return render_template('form.html')

5. Using Flask-WTF for Form Handling

Install Flask-WTF:

pip install Flask-WTF

Create a form class in forms.py:

from flask_wtf import FlaskForm
from wtforms import StringField, SubmitField
from wtforms.validators import DataRequired

class NameForm(FlaskForm):
    name = StringField('Name', validators=[DataRequired()])
    submit = SubmitField('Submit')

Use the form in your app:

@app.route('/submit_wtf', methods=['GET', 'POST'])
def submit_wtf():
    form = NameForm()
    if form.validate_on_submit():
        name = form.name.data
        return f'Thank you, {name}!'
    return render_template('form.html', form=form)

6. Database Integration with SQLAlchemy

Install SQLAlchemy and Flask-Migrate:

pip install Flask-SQLAlchemy Flask-Migrate

Configure the database:

app.config['SQLALCHEMY_DATABASE_URI'] = 'sqlite:///site.db'
db = SQLAlchemy(app)

Create a user model:

class User(db.Model):
    id = db.Column(db.Integer, primary_key=True)
    username = db.Column(db.String(20), unique=True, nullable=False)
    email = db.Column(db.String(120), unique=True, nullable=False)

7. User Authentication with Flask-Login

Install Flask-Login:

pip install Flask-Login

Add user loader and login routes:

from flask_login import LoginManager, UserMixin, login_user, logout_user, login_required

login_manager = LoginManager(app)

@login_manager.user_loader
def load_user(user_id):
    return User.query.get(int(user_id))

@app.route('/login')
def login():
    user = User.query.first()
    login_user(user)
    return 'Logged in!'

8. Building REST APIs with Flask

Install Flask-RESTful:

pip install Flask-RESTful

Add API routes:

from flask_restful import Api, Resource

api = Api(app)

class HelloWorld(Resource):
    def get(self):
        return {'message': 'Hello, World!'}

api.add_resource(HelloWorld, '/api/hello')

9. Deploying Flask to Heroku

Install gunicorn:

pip install gunicorn

Create a Procfile:

web: gunicorn app:app

Deploy to Heroku:

heroku create
git push heroku master

10. Writing Tests

Basic test example:

def test_home_page(client):
        response = client.get('/')
        assert response.status_code == 200
        assert b'Hello, World!' in response.data
